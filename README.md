# Rot-Factory
### -by Kshitij

**Rot-Factory** is a Python project to **auto-generate “brain rot” videos** from sources and upload them to YouTube.  

⚠️ **Note:** This project is in **early development**. Scripts may break, be incomplete, or incompatible across devices. Use your own knowledge to troubleshoot; AI tools can help, but nothing is guaranteed.  

---

## Features

- Auto-generates chaotic “brain rot” videos  
- Uploads videos to YouTube via Python  
- Experimental and customizable  

---

## Installation

```bash
git clone https://github.com/your-username/Rot-Factory.git
cd Rot-Factory
```

In this you can test out your creativity of own personal use only.
